# email-survey
An email-based survey application built using Flutter Web and Node JS


# Low Level Design

![low-level-design] (https://res.cloudinary.com/dek0w0qnj/image/upload/v1656691267/survey-app_lld-1_oh6v7i.png)


# Tech stack used

## Server/Backend
*Language:* JavaScript  
*Platform:* Node.js  
*Package management:* NPM
*Database:* MongoDB

*Frameworks/Packages:*
* Express JS 
* Mongoose
* Passport JS (for google oauth)
* Mailgun-JS (for sending emails from CSV data)



## Web/Frontend
*Language:* Dart 
*Framework:* Flutter

*Frameworks/Packages:*
* Provider  
* Js
* Razorpay
* Browser-Client
