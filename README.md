# email-survey
An email-based survey application built using Flutter Web and Node JS


# Tech stack used

## Server/Backend
*Language:* JavaScript  
*Platform:* Node.js  
*Package management:* NPM
*Database:* MongoDB

*Frameworks/Packages:*
* Express JS 
* Mongoose
* Passport JS (for google oauth)
* Mailgun-JS (for sending emails from CSV data)



## Web/Frontend
*Language:* Dart 
*Framework:* Flutter

*Frameworks/Packages:*
* Provider  
* Js
* Razorpay
* Browser-Client
